## 文档结构

舆情监测日报文档采用固定的结构格式，包含以下主要部分：

```
【标题】
监测时间段

【传播概述】
舆情总体情况说明

【舆情数据】
1、全网声量
2、正面声量 & 负面声量
3、评论监控

【高频正面信息】
多条新闻标题和链接

【高管高频正面信息】
多条高管相关新闻标题和链接
```

## 输入文件说明

### 1. 监测日报文件
- **格式**：.docx 文件
- **内容**：舆情监测日报正文，包含监测时段、传播概述、舆情数据、高频正面信息标题等
- **处理方式**：使用 scripts/read_docx.py 读取文档内容

### 2. 高频正面信息链接文件
- **格式**：.txt 文件
- **内容**：每行一个链接URL，按顺序对应高频正面信息的第1-5条
- **示例**：
  ```
  http://auto.ce.cn/auto/gundong/202603/t20260331_2869726.shtml
  https://www.egsea.com/news/detail/2262010.html
  https://news.mydrivers.com/1/1112/1112673.htm
  https://m.huanqiu.com/article/4QyG8F96Bqt
  https://chejiahao.autohome.com.cn/info/25105486
  ```
- **处理方式**：直接读取文本文件，按行提取链接

### 3. 输出文件
- **格式**：.txt 文件
- **命名**：舆情监测日报_[日期].txt
- **内容**：按照模板格式生成的完整文字描述

## 字段定义

### 1. 标题
- **格式**：【MM月DD日机关重点舆情监测日报】
- **示例**：【03月31日机关重点舆情监测日报】
- **提取规则**：根据监测时段日期生成标题

### 2. 监测时间段
- **格式**：MM月DD日HH:MM-MM月DD日HH:MM
- **示例**：03月30日18:00-03月31日19:00
- **提取规则**：提取监测时段字段
  - 正面新闻主要话题（多个标题，用引号分隔）
  - 负面新闻主要话题（多个标题，用引号分隔）

**提取规则**：
- 总声量：提取"报道XX篇次"中的数字
- 正面声量：提取"正面/中性XX篇"中的数字
- 正面占比：提取"占比XX%"中的数字
- 负面声量：提取"负面XX篇"中的数字
- 负面占比：提取"占XX%"中的数字
- 正面话题：提取引号内的标题内容
- 负面话题：提取引号内的标题内容

### 4. 舆情数据

#### 4.1 全网声量
- **格式**：全网关于[主体]相关报道为[数字]频次
- **提取规则**：提取数字

#### 4.2 正面声量 & 负面声量
- **格式**：正面/中性信息[数字]频次，占[百分比]%。负面声量：负面信息[数字]频次，占[百分比]%
- **提取规则**：提取两个数字和两个百分比

#### 4.3 评论监控
- **格式**：用户评论共[数字]条，正面/中性占比[百分比]%（[数字]条），负面占比[百分比]%（[数字]条），[描述]
- **提取规则**：提取评论总数、正面占比、正面评论数、负面占比、负面评论数、是否有集中吐槽内容

### 5. 高频正面信息
- **格式**：
  ```
  序号⃣[标题]
  [链接]
  ```
- **示例**：
  ```
  1⃣华为余承东谈移动卫星通信新突破，称其正逐步成为智能设备的标配
  https://www.ithome.com/0/934/209.htm
  ```
- **提取规则**：
  - 识别"【高频正面信息】"标记
  - 提取该部分所有标题和对应的链接
  - 序号格式为：数字⃣

### 6. 高管高频正面信息
- **格式**：与高频正面信息相同
- **提取规则**：
  - 识别"【高管高频正面信息】"标记
  - 提取该部分所有标题和对应的链接

## 提取规则

### 数字提取
- 使用正则表达式提取数字：`\d+`
- 百分比提取：`(\d+(?:\.\d+)?)%`

### 话题提取
- 话题通常在引号内：`"([^"]+)"`
- 多个话题用"、"分隔
- 提取时保持原文表述

### 链接提取
- 链接格式：`https?://[^\s]+`
- 每个标题后紧跟对应的链接

### 特殊情况处理
- 如果某个字段缺失，标注"未提供"
- 如果数字格式不标准，保持原文
- 如果话题或链接缺失，跳过该项

## 示例说明

### 完整示例

**输入文档内容**：
```
【0330监测日报】机关重点舆情监测日报
03月29日18:00-03月30日19:00

传播概述：全网关于Y总、H总相关报道9812篇次，正面/中性9198篇占比93.74%；负面614篇占6.26%。正面新闻主要聚焦："华为余承东谈移动卫星通信新突破，称其正逐步成为智能设备的标配"、"问界M6全系标配公布：含896线激光雷达、途灵平台与巨鲸电池"等话题。负面新闻主要聚焦："鸿蒙智驾车辆与幼童碰撞未制动"等相关敏感内容。

【舆情数据】
1、全网声量：全网关于Y总、H总相关报道为9812频次。
2、正面声量：正面/中性信息9198频次，占93.74%。负面声量：负面信息614频次，占6.26%。
3、评论监控：监测时间段内，用户评论共195824条，正面/中性占比92.58%（181300条），负面占比7.42%（14524条），未出现用户集中吐槽内容。

【高频正面信息】
1⃣华为余承东谈移动卫星通信新突破，称其正逐步成为智能设备的标配
https://www.ithome.com/0/934/209.htm

【高管高频正面信息】
1⃣华为余承东谈移动卫星通信新突破，称其正逐步成为智能设备的标配
https://www.ithome.com/0/934/209.htm
```

**提取结果**：
- 标题：【0330监测日报】机关重点舆情监测日报
- 监测时间段：03月29日18:00-03月30日19:00
- 主体：Y总、H总
- 总声量：9812
- 正面声量：9198
- 正面占比：93.74%
- 负面声量：614
- 负面占比：6.26%
- 正面话题：
  1. "华为余承东谈移动卫星通信新突破，称其正逐步成为智能设备的标配"
  2. "问界M6全系标配公布：含896线激光雷达、途灵平台与巨鲸电池"
- 负面话题：
  1. "鸿蒙智驾车辆与幼童碰撞未制动"
- 评论总数：195824
- 正面评论占比：92.58%
- 正面评论数：181300
- 负面评论占比：7.42%
- 负面评论数：14524
- 集中吐槽情况：未出现用户集中吐槽内容
- 高频正面信息：1条（标题+链接）
- 高管高频正面信息：1条（标题+链接）

## 注意事项

1. **数据准确性**：确保提取的数字和百分比与原文完全一致
2. **格式保持**：生成文字描述时保持原文的专业表述风格
3. **完整性检查**：检查是否所有字段都已提取，避免遗漏
4. **特殊字符处理**：保留原文中的特殊符号（如⃣）
5. **链接完整性**：确保链接完整无缺失
